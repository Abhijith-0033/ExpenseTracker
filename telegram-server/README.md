# Gastos Expense Tracker — Telegram Bot Backend Server

This is the standalone Node.js Express server that handles incoming webhooks from the Telegram Bot API, parses expense messages, maintains a queue of pending transactions/queries, and handles communication with the Gastos mobile app.

---

## Setup & Deployment Guide

Follow these steps to set up and deploy the Telegram Bot for your Gastos app:

### Step 1: Create a Telegram Bot
1. Open Telegram and search for [@BotFather](https://t.me/BotFather).
2. Start a chat and send the `/newbot` command.
3. Choose a name and a unique username for your bot.
4. Save the **HTTP API Token** generated by BotFather. This is your `TELEGRAM_BOT_TOKEN`.

### Step 2: Deploy to Railway.app (or any hosting provider)
1. Fork or upload this codebase to your GitHub.
2. Log in to [Railway.app](https://railway.app/) and create a **New Project**.
3. Select **Deploy from GitHub repository** and select your repository.
4. Add the following **Environment Variables** in Railway settings under "Variables":
   - `TELEGRAM_BOT_TOKEN`: The API token from Step 1.
   - `APP_SECRET_KEY`: A secure random password (min 32 characters) used to secure communication between the app and this server.
   - `PORT`: `3000` (Railway sets this automatically).
   - `RAILWAY_URL`: The public domain URL generated for your service (e.g., `https://your-service-name.up.railway.app`).
5. Deploy the server.

### Step 3: Register the Webhook
Once the server is deployed on Railway, the server will automatically attempt to register the webhook with Telegram during startup using your `RAILWAY_URL`.
To verify it, visit:
`https://your-service-name.up.railway.app/health`
It should return a JSON response with status `ok`.

### Step 4: Link to Gastos App
1. Open the Gastos App on your phone.
2. Go to **Settings** → **Telegram Bot**.
3. Enter your deployed server's URL (e.g. `https://your-service-name.up.railway.app`).
4. Enter the `APP_SECRET_KEY` that you set on your server in `telegram/TelegramService.ts` (`REPLACE_WITH_YOUR_APP_SECRET_KEY`).
5. In Telegram, start the bot by clicking **Start** or typing `/start`. Copy the connection code shown.
6. Paste the connection code in the Gastos App under Telegram settings and tap **Connect**.

## Message Format & Casual Parsing

Once linked, you can add transactions by messaging the bot directly.

### Examples:
- `food 150` -> Adds expense of ₹150 under Food.
- `transport 50 office cab` -> Adds expense of ₹50 under Transport with description "office cab".
- `income 45000 salary` -> Adds income of ₹45,000 under Income with description "salary".
- `food 150 yesterday` -> Adds expense of ₹150 under Food with yesterday's date.
- `rent 10000 05/06` -> Adds expense of ₹10,000 under Housing on 5th of June.

### Bot Commands:
- `/today` — Today's transactions summary.
- `/week` — This week's summary.
- `/month` — This month's summary.
- `/balance` — Check your account balances.
- `/categories` — List all keywords mapped to categories.
- `/undo` — Delete the last added transaction.
- `/status` — View connection status.
- `/help` — Show help message.
